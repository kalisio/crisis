# Installation

# Web application

The web application is available on https://aktnmap.kalisio.com (opens new window).

It has been tested on the following browsers:

  • Google Chrome (opens new window)
  • Mozilla Firefox (opens new window)
  • Microsoft Edge (opens new window)

TIP

We advise you to keep your browser up-to-date in order to use a reliable and secure version.

The web application does not require any installation workflow.

# Mobiles applications

You are a fieldworker ? Android (opens new window) and iOS (opens new window) apps can help you monitor and capture events on the go.

Use mobile app notifications to keep tabs on work that needs your attention and take action faster right from the app.

# Android

The Android application is available to install on Google Play (opens new window)

# iOS

The iOS application is available to install on Apple Store (opens new window).
