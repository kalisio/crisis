# Introduction

Akt'n'Map is a situational intelligence solution to inform and coordinate fieldworkers during events like interventions, crisis management, etc.

Akt'n'Map is available as:

  • a web application on https://aktnmap.kalisio.com (opens new window),
  • a mobile Android application on Google Play (opens new window),
  • a mobile iOS application on Apple Store (opens new window).

Once registered to the application you can access all versions.

# Service availability

The service availability is measured as the percentage of time the application is available over a given period. Check https://status.kalisio.com (opens new window) any time to get current status information.
