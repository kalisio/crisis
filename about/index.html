# About

Akt'n'Map is an Open Source project available on GitHub (opens new window).

It also relies on a development and deployment stack composed of:

  • KDK (opens new window), a development kit to simplify the development of geospatial applications
  • Kano (opens new window), a powerful 2D/3D geospatial data visualizer
  • Weacast (opens new window), weather forecast data management platform
  • Kargo (opens new window), a solution to deploy geospatial services
  • Kaabah (opens new window), a solution to build and operate infrastructures
