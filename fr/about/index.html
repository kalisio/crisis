# A propos

Akt'n'Map est un projet libre disponible sur GitHub (opens new window).

Il repose sur un écosystème de développement et de déploiement composé de:

  • KDK (opens new window), un kit de développement pour les applications géospatiales
  • Kano (opens new window), un visualiseur 2D/3D de données géospatiales
  • Weacast (opens new window), une plateforme de gestion de données météorologiques,
  • Kargo (opens new window), une solution pour déployer des services géospatiaux
  • Kaabah (opens new window), une solution pour déployer et opérer des infrastructures
