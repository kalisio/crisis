# Didacticiels Vidéo

Cette section présente l'essentiel des fonctionnalités d'Akt'n'Map grâce à des didacticiels ou tutoriels vidéo. Vous pouvez retrouver l'ensemble de ces vidéos sur notre playlist (opens new window).

Attention

Akt'n'Map s'améliore et évolue de façon continue. Aussi, ce qui s’affiche sur votre écran peut parfois différer de ce que vous verrez dans ces vidéos, mais les concepts clés restent les mêmes.

# Gérez vos événements
